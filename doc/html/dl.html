pdnsd Download Page

To obtain pdnsd from the original author, please visit http://home.t-online.de/home/Moestl/dl.html
Note that pdnsd is no longer maintained by Thomas Moestl.
However, Paul Rombouts maintains a version with many fixes and improvements at http://www.phys.uu.nl/~rombouts/pdnsd.html.
He has pre-patched tarballs and RPM packages available for download at this site.
There is also a mirror site at http://members.home.nl/p.a.rombouts/pdnsd.html.

The most recent tarball is pdnsd-1.2.4-par.tar.gz (GPG signature).
The most recent RPM packages are:
Package Name Size Description
pdnsd-1.2.4-par.src.rpm 532kB Source RPM (binary packages can also be built directly from the tarball).
pdnsd-1.2.4-par_sl4.i386.rpm 284kB i386 binary built on a Scientific Linux 4 system (compatible with Red Hat EL 4).
pdnsd-1.2.4-par_fc4.x86_64.rpm 298kB x86_64 binary built on a Fedora Core 4 system.
If you want to check the signatures on these packages you will need a copy of my GPG key which you can get here or from a public key server.

There are also Debian and Gentoo packages, but these may not include the latest version.
If you are looking for other versions released by Paul Rombouts, visit this download directory.

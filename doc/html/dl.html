pdnsd Download Page

To obtain pdnsd from the original author, please visit http://home.t-online.de/home/Moestl/dl.html
Note that pdnsd is no longer maintained by Thomas Moestl.
However, Paul Rombouts maintains a version with many fixes and improvements at http://www.phys.uu.nl/~rombouts/pdnsd.html.
He also has pre-patched tarballs and RPM packages available for download at this site.
There is also a mirror site at http://members.home.nl/p.a.rombouts/pdnsd.html.

The most recent tarball is pdnsd-1.2-par.tar.gz.
The most recent RPM packages are:
Package Name Size Description
pdnsd-1.2-par.src.rpm 344kB Source RPM (binary packages can also be built directly from the tarball).
pdnsd-1.2-par_rh9.i386.rpm 256kB i386 binary built on a Red Hat 9 system, without isdn support.
pdnsd-1.2-par_rh9_isdn.i386.rpm 256kB i386 binary built on a Red Hat 9 system, with isdn support.

There are also Debian and Gentoo packages, but these may not include the latest version.
If you are looking for other versions released by Paul Rombouts, visit this download directory.
